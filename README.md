vim-pathogen-installer
======================

Wrapper for [vim-pathogen](https://github.com/tpope/vim-pathogen) installation.

Run this script with:

    bash <(curl -sL https://github.com/phpshift/vim-pathogen-installer/raw/master/install.sh)
